# Entregables

1. URL de la aplicación deployada:
   https://app-libros-10.vercel.app/

2. Repositorio GitHub:
   https://github.com/HarrysonLadines/APPLibros

3. Documentación:

   a) Cómo hacer el deploy local:
      - Clonar el repositorio
      - Instalar dependencias con `npm install`
      - Generar Prisma Client con `npx prisma generate`
      - Correr el proyecto con `npm run dev`

   b) Cómo funcionan los GitHub Actions:
      - Cada push o pull request corre los tests con `npm test`
      - Actions verifica que el proyecto construya correctamente y que los scripts se ejecuten

   c) Variables de entorno necesarias:
      - `DATABASE_URL` para la conexión de Prisma

   d) Instrucciones para ejecutar con Docker:
      - `docker build -t mi-app .`
      - `docker run -p 3000:3000 mi-app`

4. Demostración de GitHub Actions:
   - En el repositorio se pueden ver los workflows en `.github/workflows`
   - Todas las pruebas corren, a excepción de test.yml, tuve un error de versiones al hacer push a main y se borraron archivos de tests.
